# Overview

 * The project will implement the following:
     
     - Collecting data for various types of weapons in Call of Duty Warzone 3.
     - Graphs to show the popularity/strength of weapons based on their rating.

# Technologies, Stack:
    
 * Typescript
 * Next.js + TS
 * TailwindCSS
 * GraphQL
 * amCharts 5
